# cpro-common-lib

## Overview

This is a library helm chart, which provides helper functions. It does not provide any Kubernetes objects.


## Example usage

* Add `cpro-common-lib` to the `requirements.yaml` of your helm chart.

```
dependencies:
- name: cpro-common-lib
  version: 1.0.0-1
  repository: https://repo.cci.nokia.net/csf-helm-inprogress/
```
